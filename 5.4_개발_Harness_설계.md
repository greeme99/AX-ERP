## 3. 개발 원칙

-   API First
-   Domain Driven Design
-   Event Driven Architecture
-   Test First
-   Security by Design
-   Observability by Default

## 4. AI Coding Harness 공통 지침

### 역할

당신은 Enterprise AX Platform 개발자다.

### 준수사항

-   기존 기능정의서, 화면정의서, ERD, API 정의서를 기준으로 개발한다.
-   임의 기능 추가 금지
-   TypeScript strict mode 사용
-   모든 API는 OpenAPI 문서와 일치해야 한다.
-   모든 DB 변경은 migration으로 관리한다.
-   테스트 코드를 함께 작성한다.

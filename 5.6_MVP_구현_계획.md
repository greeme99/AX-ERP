# 06. MVP 구현 계획 v1.0

## 1. MVP 목표

Enterprise AX 플랫폼 전체 중 가장 빠르게 가치를 검증할 수 있는 핵심
기능을 구현한다.

## 2. MVP 범위

-   Executive Dashboard
-   SCM Control Tower
-   Production Control Tower
-   Inventory Dashboard
-   AI Copilot
-   기본 Master Data
-   기본 Sales / Purchase / Production 데이터

## 3. MVP 기능 목록

### Dashboard

-   CEO KPI Card
-   매출 Trend
-   재고 Trend
-   생산 OEE
-   공급망 Risk

### SCM

-   Forecast 조회
-   부족자재 알림
-   공급사 납기 Risk

### Production

-   생산계획 조회
-   생산실적 조회
-   라인별 OEE

### AI Copilot

-   KPI 질의
-   원인 분석
-   리스크 요약
-   일일 브리핑 생성

## 4. MVP 기술 스택

-   Frontend: Next.js, React, TypeScript
-   Backend: FastAPI 또는 Spring Boot
-   DB: PostgreSQL
-   AI: LangGraph, OpenAI API 또는 사내 LLM
-   Infra: Docker Compose

## 5. MVP 일정

-   Week 1: 데이터 모델 및 Mock API
-   Week 2: Dashboard UI
-   Week 3: SCM/Production 화면
-   Week 4: AI Copilot
-   Week 5: 통합 테스트
-   Week 6: 데모 및 피드백

## 6. MVP 성공 기준

-   CEO Dashboard 시연 가능
-   SCM Risk 탐지 가능
-   생산 OEE 조회 가능
-   AI Copilot 질의응답 가능
-   사용자 피드백 80점 이상

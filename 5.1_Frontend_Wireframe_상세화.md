# 01. Front-end Wireframe 상세화 v1.0

## 1. 목적

AX 통합 플랫폼의 주요 화면 구조를 실제 개발 가능한 Wireframe 수준으로
정의한다.

## 2. 공통 화면 레이아웃

``` text
┌─────────────────────────────────────────────────────────────┐
│ Header: Logo | Global Search | Alerts | Language | User      │
├───────────────┬───────────────────────────────┬─────────────┤
│ Left GNB      │ Main Workspace                 │ AI Copilot  │
│ - Dashboard   │ - Page Title                   │ - 질문 입력 │
│ - Sales       │ - Filter Area                  │ - 분석 결과 │
│ - SCM         │ - KPI Cards                    │ - 추천 액션 │
│ - Production  │ - Data Grid / Chart / Form     │             │
│ - Quality     │                               │             │
│ - Finance     │                               │             │
└───────────────┴───────────────────────────────┴─────────────┘
```

-   상단 KPI Card 8개
-   중단: 매출/손익 Trend, 글로벌 Map
-   하단: 리스크 Top 5, AI Executive Briefing

### 주요 컴포넌트

-   KPI Card
-   Trend Chart
-   Risk Alert Card
-   AI Briefing Panel

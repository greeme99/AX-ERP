# 3. 기능 정의

## SD-001 고객 마스터 관리

### 목적

고객 기본정보 및 거래정보 관리

### 주요 기능

-   고객 등록
-   고객 수정
-   거래중지 처리
-   고객 등급관리
-   고객 신용한도 관리

### 입력항목

-   고객코드
-   고객명
-   국가
-   통화
-   결제조건

### 출력

-   고객마스터 조회
-   거래현황

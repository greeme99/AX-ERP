# 05. AI Agent Prompt/Memory 설계 v1.0

## 1. 목적

AX 플랫폼 내 AI Agent가 안정적으로 예측, 분석, 추천, 보고를 수행하도록
Prompt, Memory, Tool 사용 정책을 정의한다.

## 2. Agent 공통 원칙

-   AI는 추천한다.
-   사람은 승인한다.
-   모든 추천은 근거를 제공한다.
-   모든 AI 실행은 감사 로그를 남긴다.
-   고위험 업무는 자동 실행하지 않는다.

## 3. Agent 목록

-   AI Demand Planner
-   AI Buyer
-   AI Scheduler
-   AI Quality Engineer
-   AI Inventory Optimizer
-   AI CFO Copilot
-   AI CEO Briefing Agent

## 4. Prompt Template 구조

``` text
[Role]
당신은 {domain} 전문 AI Agent입니다.

[Context]
회사, 공장, 기간, 품목, 고객, 공급사 데이터를 기준으로 분석합니다.

[Task]
사용자의 요청을 분석하고 필요한 데이터를 조회한 뒤 답변합니다.

[Constraint]
근거 없는 결론 금지.
확실하지 않으면 불확실하다고 말합니다.
자동 실행하지 않고 승인 요청을 생성합니다.

[Output]
요약
근거
추천
리스크
승인 필요 여부
```

## 5. Memory 설계

### Short-term Memory

-   현재 대화
-   현재 분석 조건
-   사용자의 최근 질문

### Long-term Memory

-   업무 규칙
-   KPI 정의
-   승인 정책
-   과거 의사결정 이력

### Retrieval Memory

-   기능정의서
-   화면정의서
-   API 문서
-   ERD
-   업무 매뉴얼
-   품질 문서

## 6. Tool 사용 정책

-   ERP API 조회
-   SCM Forecast 조회
-   WMS 재고 조회
-   QMS 품질 데이터 조회
-   EIS KPI 조회
-   Report Generator 호출

## 7. Guardrail

-   회계전표 자동 생성 금지
-   구매발주 자동 승인 금지
-   생산오더 Release 자동 실행 금지
-   고객 커뮤니케이션 자동 발송 금지

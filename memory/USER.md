# USER (AX-ERP 프로젝트)

§ 사용자(그리메)는 경영/기획/전략/오퍼레이션 최적화 컨설턴트. 마케팅/영업/인사/개발/품질/SCM/물류/구매/제조/생산관리 등 다기능 실무 경험 보유.
§ 최근 3년 생성형 AI 전문가로 AI 모델/Agent/오픈소스/노코드·로우코드·바이브코딩 서비스 기획·개발.
§ 응답 스타일 선호: 간결·직접적, 불필요한 설명 최소화. 각 직무 전문가 관점의 체계적 가이드, 이론-실무 연결된 실행 가능한 조언 선호.
§ 개발 방식 선호: 에러 최소화, 토큰 사용 최소화, 프로토타입(v1~v2) 먼저 검증 후 단계적 구체화.

# ERP 기능정의서 v3.4

# Workflow 정의서 (Enterprise Level)

## 문서 목적

본 문서는 Enterprise ERP 운영을 위한 표준 Workflow를 정의한다.

대상 시스템: - PLM - ERP - MES - WMS - QMS - SCM - EIS - AI Agent

------------------------------------------------------------------------

# 1. 영업(SD) Workflow

## WF-SD-001 견적 승인 프로세스

영업담당자 → 영업팀장 검토 → 사업부장 승인 → 고객 제출 → 수주 전환

승인 기준 - 5천만원 미만 : 팀장 승인 - 5천만원 이상 : 사업부장 승인 -
5억원 이상 : 임원 승인

------------------------------------------------------------------------

## WF-SD-002 수주 승인 프로세스

수주 등록 → ATP 확인 → 신용한도 확인 → 생산가능 여부 확인 → 수주 승인 →
생산계획 연계

예외 처리 - 신용한도 초과 - 재고 부족 - 생산능력 부족

------------------------------------------------------------------------

# 2. 구매(MM) Workflow

## WF-MM-001 구매요청(PR)

요청자 → 부서장 승인 → 구매팀 검토 → 구매발주 생성

승인 금액 기준 - 1천만원 미만 : 부서장 - 1억원 미만 : 본부장 - 1억원
이상 : 임원

------------------------------------------------------------------------

## WF-MM-002 구매발주(PO)

구매발주 생성 → 공급사 선정 → 발주 승인 → 공급사 송부 → 납기 모니터링 →
입고 연계

------------------------------------------------------------------------

# 3. 생산(PP) Workflow

## WF-PP-001 생산계획

Forecast → S&OP → MPS → Capacity Check → 생산계획 승인 → MRP 실행

------------------------------------------------------------------------

## WF-PP-002 생산오더

생산오더 생성 → 작업지시 → 자재출고 → 생산실행 → 실적수집 → 생산마감

------------------------------------------------------------------------

# 4. 품질(QM) Workflow

## WF-QM-001 수입검사

입고 → 검사대기 → 검사수행 → 합격/불합격 판정 → 사용승인 또는 반송

------------------------------------------------------------------------

## WF-QM-002 고객클레임

클레임 접수 → 원인분석 → 8D Report → CAPA → 효과검증 → 종료

------------------------------------------------------------------------

# 5. 물류(WMS) Workflow

## WF-LE-001 입고 프로세스

ASN 수신 → 입고 → 검수 → Putaway → 재고반영

------------------------------------------------------------------------

## WF-LE-002 출고 프로세스

출하지시 → Picking → Packing → Shipping → 매출연계

------------------------------------------------------------------------

# 6. 재무(FI) Workflow

## WF-FI-001 월마감

전표마감 → 재고마감 → 원가계산 → 손익계산 → 결산보고 → 마감승인

------------------------------------------------------------------------

# 7. PLM Workflow

## WF-PLM-001 ECO

변경요청 → 영향분석 → 승인 → BOM 변경 → 생산반영

------------------------------------------------------------------------

# 8. SCM Workflow

## WF-SCM-001 S&OP

Forecast 생성 → 영업 검토 → 생산 검토 → 구매 검토 → 경영진 승인 →
실행계획 확정

------------------------------------------------------------------------

# 9. AI Agent Workflow

## WF-AI-001 AI 추천 프로세스

데이터 수집 → AI 분석 → 추천 생성 → 사용자 검토 → 승인 → 실행

원칙 - AI는 자동 실행하지 않는다. - 승인 기반 실행을 원칙으로 한다.

------------------------------------------------------------------------

## WF-AI-002 CEO Briefing

ERP 데이터 수집 → KPI 생성 → 리스크 분석 → AI 요약 → CEO 보고

------------------------------------------------------------------------

# 10. Escalation Workflow

Level 1 - 담당자

Level 2 - 팀장

Level 3 - 본부장

Level 4 - 임원

적용 대상 - 납기 지연 - 품질 사고 - 공급 중단 - 생산 중단 - 보안 사고

------------------------------------------------------------------------

# 11. Global Workflow

Local Approval → Regional Approval → HQ Approval → Global Execution

적용: - CAPEX - Global Sourcing - 신규 고객 승인 - 신규 공장 승인

------------------------------------------------------------------------

# 12. Workflow KPI

-   승인 Lead Time
-   승인 병목률
-   자동 승인율
-   SLA 준수율
-   예외 처리시간
-   AI 추천 채택률

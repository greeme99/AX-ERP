# 07. Pilot 구축 계획 v1.0

## 1. 목적

MVP를 실제 현업 환경에 적용하여 업무 적합성, 데이터 품질, 사용자
수용성을 검증한다.

## 2. Pilot 범위

-   1개 법인
-   1개 공장
-   2\~3개 핵심 라인
-   50\~100명 사용자
-   3개월 운영

## 3. Pilot 대상 업무

-   SCM Control Tower
-   생산 실적 모니터링
-   재고 가시성
-   품질 이상 알림
-   AI Daily Briefing

## 4. Pilot 준비

-   기준정보 정비
-   품목/BOM/거래처 데이터 이관
-   사용자 권한 설정
-   교육자료 작성
-   운영 Help Desk 구성

## 5. Pilot 운영 단계

1.  Kick-off
2.  Key User 교육
3.  데이터 검증
4.  Parallel Run
5.  Go-Live
6.  Hypercare
7.  효과 분석

## 6. Pilot KPI

-   사용자 Adoption Rate
-   데이터 정확도
-   업무 처리시간 감소
-   리스크 탐지 건수
-   AI 추천 채택률
-   시스템 안정성

## 7. Pilot Exit Criteria

-   주요 업무 정상 처리
-   Critical Issue 0건
-   사용자 만족도 80점 이상
-   경영진 확산 승인

# 2. 권한 모델

## 2.1 RBAC

-   Role Based Access Control
-   역할 기반 권한 부여

## 2.2 ABAC

-   Attribute Based Access Control
-   회사, 공장, 조직, 국가, 직급 기준 권한 제어

-   Segregation of Duties

# 7. AI Agent 권한 정책

## AI Demand Planner

-   Forecast 생성 가능
-   승인 권한 없음

## AI Buyer

-   구매 추천 가능
-   발주 승인 불가

## AI Scheduler

-   생산계획 추천 가능
-   생산 Release 불가

## AI CFO Copilot

-   손익 분석 가능
-   회계전표 생성 불가

원칙: AI Agent는 추천 권한만 보유하며 실행 권한은 보유하지 않는다.

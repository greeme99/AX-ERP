# 08. Global Rollout 계획 v1.0

## 1. 목적

Pilot 성공 후 글로벌 법인과 공장으로 AX 플랫폼을 단계적으로 확산한다.

## 2. Rollout 원칙

-   Global Template 우선
-   Local Requirement는 Add-on으로 관리
-   Core Customizing 최소화
-   Data Governance 선행
-   Change Management 병행

## 3. Rollout Wave

### Wave 1

-   Korea HQ
-   핵심 ERP Core
-   EIS Dashboard

### Wave 2

-   Vietnam Plant
-   MES/WMS/QMS 연계
-   생산/품질 중심

### Wave 3

-   China Plant
-   구매/생산/물류 확장

### Wave 4

-   Mexico Plant
-   글로벌 공급망 대응

### Wave 5

-   Sales Subsidiaries
-   영업/서비스/재무 확산

## 4. Global Template 관리

-   Global Process Owner 지정
-   Local Process Owner 지정
-   Change Control Board 운영
-   Localization Rule 관리

## 5. Change Management

-   임원 Sponsor 확보
-   Key User Network 구축
-   교육/워크샵
-   현장 저항 관리
-   성과 공유

## 6. Rollout KPI

-   Template Fit Rate
-   Localization Ratio
-   Go-Live Success Rate
-   Issue Resolution Time
-   User Adoption
-   Business Benefit

## 7. 최종 목표

Global AX Operating Model을 구축하여 전 세계 법인, 공장, 고객, 공급망을
하나의 데이터와 AI 기반 운영체계로 연결한다.
